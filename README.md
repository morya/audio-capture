# 主体任务

帮我用 python + pyside6 + pyaudio 编写一个简单的麦克风录音和渲染工具。

## 分步骤任务

1. 搭建 pyside6 界面，包含录音按钮、渲染区域等。
2. 实现录音功能，使用 pyaudio 库捕获麦克风输入。
    2.1 给出麦克风按钮，并且，按钮旁边有一个下拉菜单小三角，可以通过下拉菜单动态的读取到当前的麦克风列表，
    并可以通过下拉菜单选择当前准备使用的麦克风。
3. 实时渲染录音数据，将其绘制到渲染区域。
4. 添加播放功能，使用 pyaudio 库播放渲染区域的音频数据。

## 开发支持部分

- 生成 github actions 工作流，用于自动构建和发布项目。
- 需要支持 windows 10 x64, macos 10.15 arm64 以上版本。
- 不需要支持 linux 系统。
- 不支持 macos intel cpu 版本。

## 注意事项

1. 确保系统已安装 pyside6、pyaudio 库。
2. 运行程序时，需要自动判断当前操作系统针对麦克风设备的授权要求，
   如果没有授权，需要提示用户授权。
3. 连接麦克风设备。
